# ai-audit

A periodic **review of the project's AI surface** (rules, skills, agents,
commands): duplicates, stale instructions, `alwaysApply` bloat, and where each
piece belongs.

**Process, checklists, and report format** — strictly per
**`docs/ai/recipes/ai-audit.md`** (read it and follow it).

Do not apply edits yourself without an explicit user request: produce a report
from the recipe's template and concrete proposals (`delete` / `condense` /
`move`).
