# Core Profile

Always active. Follow the stack-neutral rules and workflows in `docs/ai/` plus
the repository-specific constraints in `docs/ai/rules/coding-rules.md`.

The core assumes only source files, observable behavior, project boundaries,
verification, and Git when a Git-specific workflow is invoked. It does not assume
a programming language, UI framework, network architecture, authentication
scheme, package manager, or product domain.
