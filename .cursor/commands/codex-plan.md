# codex-plan

Orchestrate planning through the Codex CLI: close questions, have Codex write the
plan, reconcile it, and record the artifact. You do not write the plan yourself.

Follow `docs/ai/commands/codex-plan.md` exactly. Use `docs/ai/agents/cli.md` for
CLI calls, `docs/ai/skills/plan.md` for plan content, and
`docs/ai/skills/implementation-clarify.md` for the question loop. Write the plan to
`docs/ai/plans/YYYY-MM-DD-short-task.plan.md`.
